Overview

What Are Claude Skills?

Claude Skills are reusable knowledge modules that extend Claude with domain-specific expertise. When a user's task matches a skill's trigger conditions, Claude automatically invokes it—no manual selection required.

These skills are production-ready: they include feasibility checks, validation, error handling, and integration with upstream tooling. They turn Claude into a specialized assistant for network test automation.
